# Sistem Manajemen Dokumen (CLI) - Python
Repositori ini berisi *source code* untuk tugas akhir Project-Based Learning (PBL) mata kuliah Algoritma & Struktur Data (TPL2106), Sekolah Vokasi IPB University.

Aplikasi ini adalah sistem digital perantara untuk mengelola tumpukan dokumen meja kerja fisik. Sistem ini diimplementasikan menggunakan struktur data **Stack (LIFO)** untuk memastikan dokumen yang paling baru masuk akan diproses terlebih dahulu. Antarmuka aplikasi dibangun menggunakan *library* **Textual** untuk menghadirkan pengalaman *Text User Interface* (TUI) di dalam terminal.

## Fitur Utama & Algoritma
* 
**Penyimpanan Permanen (I/O):** Menggunakan *file handling* teks datar (`data.txt` & `riwayat.txt`) tanpa RDBMS.
* 
**Eksekusi Stack (LIFO):** Implementasi fungsi *pop()* dengan kompleksitas waktu amortisasi O(1) untuk penarikan dokumen teratas.
* 
**Searching:** Pencarian data berkas menggunakan algoritma *Sequential Search* dengan pencocokan substruktur *string*.
* 
**Sorting:** Pengurutan tumpukan berdasarkan kronologis waktu dan alfabet menggunakan algoritma *Merge Sort* (O(N log N)).
